#  Titanic EDA Project

## Overview
Exploratory Data Analysis on the Titanic dataset
using Python, Pandas and Matplotlib.

## What I did
- Loaded real world data from URL
- Inspected shape, dtypes, missing values
- Handled missing values (Age→median, Embarked→mode, Cabin→dropped)
- Detected outliers using IQR method on all numerical columns
- Visualized data using Histogram, Boxplot, Bar chart, Scatter plot
- Clipped Fare outliers at 65.63
- Normalized Age and Fare columns
- Drew meaningful conclusions

## Key Findings
- Survival Rate: 38.4%
- Most passengers were aged 20-35
- Fare had 116 outliers — clipped at 65.63
- Age and Fare are weakly correlated
